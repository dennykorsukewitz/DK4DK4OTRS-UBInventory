# Funktionalität

Diese Erweiterung ermöglicht eine einfache Inventur via Database-Storage.

## Tabellen

Folgende Tabellen werden dafür verwendet:

- ID
- type
- model
- manufacturer
- serialnumber
- purchase_time
- comment
- create_time
- create_by
- change_time
- change_by
- employee
- room
- phone
- sap
- ip
- mac
- socket
- distribution_cabinet
- keynr
- segregation
- segregationstatus
